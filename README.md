WalkSee is an advanced software engineering project designed to empower visually impaired individuals with independent and safe urban navigation using an AI-assisted smart cane.  The system runs locally and processes real-time sensor and video feeds:  Safe Path Segmentation: Uses PIDNet (custom-trained on local Israeli street imagery) to identify walkable sidewalk boundaries in real-time.  Obstacle Detection: Employs YOLOv8-seg for precise object detection and instance segmentation, filtering out obstacles outside the walkable path.  Motion Prediction: Integrates a Kalman Filter to track and predict the trajectories of dynamic obstacles (pedestrians, cyclists).
